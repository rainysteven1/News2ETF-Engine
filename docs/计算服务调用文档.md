# 计算服务 OpenAPI 调用指南

## 文档概览

这是一份关于计算服务平台 OpenAPI 的调用规范文档，为客户端和第三方开发者提供接口调用参考。

---

## 核心调用机制

### 1. 认证流程

- **认证方式**：基于 Token 的用户认证机制
- **第一步**：调用登录接口获取 Token 凭证
- **后续步骤**：在所有其他接口请求的 HTTP 请求头中携带 Token

### 2. 权限要求

- 需要获取授权的 **orgId**（组织ID）
- 用户名注册规则：`第三方标识_用户名`（例如：`xxx_test`）
- 联系计算服务运营管理员进行授权

### 3. 通信协议

- **协议**：HTTPS（确保数据加密传输）
- **字符编码**：UTF-8
- **Content-Type**：application/json;charset=UTF-8

---

## API 调用规范

### 接口 URL 格式

接口定义中的 URL 需要补全前缀和后缀：

**示例 1**：

```
接口定义：GET /ac/openapi/v2/user
实际调用：https://[服务器地址]/ac/openapi/v2/user
```

**示例 2**：

```
接口定义：GET /jm_usermanagement/api/userquotas/queues
实际调用：https://[服务器地址]/jm_usermanagement/api/userquotas/queues
```

### 请求头设置

```
Authorization: Bearer [token]
Content-Type: application/json;charset=UTF-8
Accept-Language: zh-CN,zh;q=0.8,zh;q=0.6
```

**注意**：除了登录接口外，所有其他接口都需要在请求头中设置 `token` 字段。

---

## 返回结果格式

### 成功响应（HTTP 200）

**标准格式**：

```json
{
  "code": "0",
  "msg": "success",
  "data": {
    // JSON 对象或列表
  }
}
```

### 错误响应

**标准格式**：

```json
{
  "code": "非零值",
  "msg": "错误信息描述",
  "data": ""
}
```

### 特殊情况：Token 过期

```json
{
  "msg": "token is expired",
  "code": "10008",
  "data": null
}
```

**处理方式**：当收到此响应时，需要重新调用登录接口获取新的 Token。

---

## 错误处理流程

### 第一步：检查 HTTP 状态码

- 如果返回 404、500 等非 200 状态码，按照标准 HTTP 错误处理

### 第二步：检查响应体

- 当 HTTP 状态码为 200 时，检查返回结果中的 `code` 字段
- 根据 `code` 值判断是否成功（0 表示成功，非 0 表示失败）

### 第三步：特殊处理

- 如果返回结果不是 JSON 格式，视为错误
- 如果返回结果中缺少某个属性，将其视为空字符串处理

---

## 多语言支持

返回结果的多语言信息由请求头中的 `Accept-Language` 字段决定：

```
Accept-Language: zh-CN,zh;q=0.8,zh;q=0.6
```

- 默认返回中文
- 多语化处理在客户端进行

---

## 调用流程总结

```
1. 获取 orgId 授权
   ↓
2. 注册用户（用户名格式：第三方标识_用户名）
   ↓
3. 调用登录接口获取 Token
   ↓
4. 在请求头中携带 Token
   ↓
5. 调用具体业务接口
   ↓
6. 检查响应状态码和 code 字段
   ↓
7. 如果 Token 过期（code=10008），重新获取 Token
```

# 三大核心 API 调用指南

## 一、作业管理 API (Job Manager)

### 1. 提交作业

**端点**: `POST /hpc/openapi/v2/apptemplates/BASIC/BASE/job`

**请求头**:

```
Authorization: Bearer [token]
Content-Type: application/json;charset=UTF-8
```

**请求体参数**:

```json
{
  "strJobManagerID": 1,
  "mapAppJobInfo": {
    "GAP_CMD_FILE": "command to execute",
    "GAP_NNODE": 1,
    "GAP_NODE_STRING": "node_name",
    "GAP_SUBMIT_TYPE": "cmd",
    "GAP_JOB_NAME": "job_name",
    "GAP_WORK_DIR": "/path/to/workdir",
    "GAP_QUEUE": "queue_name",
    "GAP_NPROC": 4,
    "GAP_NGPU": 1,
    "GAP_NDCU": 0,
    "GAP_WALL_TIME": "01:00:00",
    "GAP_APPNAME": "app_name",
    "GAP_STD_OUT_FILE": "/path/to/stdout",
    "GAP_STD_ERR_FILE": "/path/to/stderr"
  }
}
```

**响应格式**:

```json
{
  "code": "0",
  "msg": "success",
  "data": "job_id_12345"
}
```

**关键参数说明**:

- `strJobManagerID`: 调度器 ID
- `GAP_CMD_FILE`: 要执行的命令
- `GAP_NNODE`: 节点数量
- `GAP_NPROC`: CPU 核心数
- `GAP_NGPU`: GPU 卡数
- `GAP_WALL_TIME`: 最大运行时间（格式：HH:MM:ss）
- `GAP_QUEUE`: 队列名称

**错误码**:

- `0`: 成功
- `10001`: 内部错误
- `10003`: 参数不完整
- `10004`: 参数无效

## 二、文件传输 API (File Transfer)

### 1. 列表文件

**端点**: `GET /efile/openapi/v2/file/list`

**请求头**:

```
Authorization: Bearer [token]
```

**查询参数**:

```
limit=10          # 页面大小，默认 10
start=0           # 起始索引，默认 0
path=/home/user   # 目标文件夹路径（绝对路径）
order=desc        # 排序顺序：asc 或 desc
orderBy=name      # 排序字段：name、size、lastModifiedTime
keyWord=search    # 搜索关键词（可选）
```

**响应格式**:

```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 100,
    "fileList": [
      {
        "name": "file.txt",
        "path": "/home/user/file.txt",
        "size": 1024,
        "permissions": "rw-r--r--",
        "lastModifiedTime": 1234567890,
        "owner": "username"
      }
    ],
    "children": [
      {
        "id": "folder_id",
        "label": "folder_name",
        "path": "/home/user/folder"
      }
    ],
    "path": "/home/user"
  }
}
```

**错误码**:

- `911020`: 文件不存在
- `911030`: 权限不足
- `911501`: 存储空间满

### 2. 上传文件

**端点**: `POST /efile/openapi/v2/file/upload`

**请求头**:

```
Authorization: Bearer [token]
Content-Type: multipart/form-data
```

**表单参数**:

```
path: /home/user/          # 目标目录路径
file: [binary file data]   # 要上传的文件
cover: cover               # 可选：cover 覆盖，uncover 跳过（默认）
```

**响应格式**:

```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```

**错误码**:

- `911020`: 文件不存在
- `911021`: 文件已存在
- `911030`: 权限不足
- `911501`: 存储空间超限
- `911700`: 文件名包含非法字符

### 3. 下载文件

**端点**: `GET /efile/openapi/v2/file/download`

**请求头**:

```
Authorization: Bearer [token]
Range: bytes=0-2000        # 可选：用于断点续传
```

**查询参数**:

```
path=/home/user/file.txt   # 文件或文件夹路径
```

**响应**:

- 返回二进制文件流
- 文件夹返回 zip 压缩流
- 支持 HTTP Range 头进行断点续传

**错误码**:

- `911020`: 文件不存在
- `911030`: 权限不足
- `911504`: 命令执行超时

## 三、容器管理 API (Container Manager)

### 1. 创建容器实例

**端点**: `POST /ai/openapi/v2/instance-service/task`

**请求头**:

```
Authorization: Bearer [token]
Content-Type: application/json;charset=UTF-8
```

**请求体参数**:

```json
{
  "instanceServiceName": "my-container",
  "taskType": "jupyter",
  "acceleratorType": "gpu",
  "imagePath": "registry.example.com/image:latest",
  "cpuNumber": 4,
  "ramSize": 8192,
  "gpuNumber": 1,
  "resourceGroup": "default",
  "taskNumber": 1,
  "timeoutLimit": "12:00:00",
  "version": "latest",
  "description": "My container instance",
  "useStartScript": true,
  "startScriptContent": "#!/bin/bash\necho 'Starting container'",
  "mountInfoList": [
    {
      "sourcePath": "/data",
      "targetPath": "/mnt/data",
      "accessMode": "rw"
    }
  ],
  "containerPortInfoList": [
    {
      "containerPort": 8888,
      "protocol": "tcp"
    }
  ]
}
```

**响应格式**:

```json
{
  "code": "0",
  "msg": "success",
  "data": "task_id_abc123"
}
```

**关键参数说明**:

- `instanceServiceName`: 容器实例标识符
- `taskType`: 任务类型（ssh、jupyter、codeserver、rstudio）
- `acceleratorType`: 加速器类型（mlu、dcu、gpu、cpu）
- `imagePath`: 容器镜像注册表路径
- `cpuNumber`: CPU 核心数
- `ramSize`: 内存大小（MB）
- `gpuNumber`: GPU 卡数
- `timeoutLimit`: 自动停止时长（格式：HH:MM:SS）
- `startScriptContent`: 启动脚本（使用 `\n` 换行）

**错误码**:

- `0`: 成功
- `10001`: 内部错误
- `10003`: 参数不完整
- `10004`: 参数无效
- `716865`: 任务创建失败

## 实现建议

### Python 完整示例

```python
import requests
import json
from typing import Dict, List, Optional, Any

class ComputeServiceClient:
    """计算服务 OpenAPI 客户端"""

    def __init__(self, base_url: str, username: str, password: str):
        self.base_url = base_url
        self.username = username
        self.password = password
        self.token = None
        self.session = requests.Session()

    def login(self) -> bool:
        """获取认证 Token"""
        url = f"{self.base_url}/ac/openapi/v2/login"
        payload = {
            "username": self.username,
            "password": self.password
        }
        try:
            response = self.session.post(url, json=payload)
            if response.status_code == 200:
                data = response.json()
                if data.get("code") == "0":
                    self.token = data.get("data", {}).get("token")
                    return True
        except Exception as e:
            print(f"Login failed: {e}")
        return False

    def _get_headers(self) -> Dict[str, str]:
        """获取请求头"""
        return {
            "Authorization": f"Bearer {self.token}",
            "Content-Type": "application/json;charset=UTF-8",
            "Accept-Language": "zh-CN,zh;q=0.8,zh;q=0.6"
        }

    def _handle_response(self, response: requests.Response) -> Dict[str, Any]:
        """处理响应"""
        if response.status_code == 200:
            data = response.json()
            if data.get("code") == "10008":  # Token 过期
                self.token = None
                if self.login():
                    return {"retry": True}
            return data
        raise Exception(f"HTTP Error: {response.status_code}")

    # ===== 作业管理 API =====

    def submit_job(self, job_config: Dict[str, Any]) -> str:
        """提交作业"""
        url = f"{self.base_url}/hpc/openapi/v2/apptemplates/BASIC/BASE/job"
        headers = self._get_headers()

        response = self.session.post(url, json=job_config, headers=headers)
        result = self._handle_response(response)

        if result.get("retry"):
            return self.submit_job(job_config)

        if result.get("code") == "0":
            return result.get("data")
        raise Exception(f"Submit job failed: {result.get('msg')}")

    # ===== 文件管理 API =====

    def list_files(self, path: str = "/", limit: int = 10,
                   start: int = 0, order: str = "desc",
                   order_by: str = "name") -> Dict[str, Any]:
        """列表文件"""
        url = f"{self.base_url}/efile/openapi/v2/file/list"
        headers = self._get_headers()

        params = {
            "path": path,
            "limit": limit,
            "start": start,
            "order": order,
            "orderBy": order_by
        }

        response = self.session.get(url, headers=headers, params=params)
        result = self._handle_response(response)

        if result.get("retry"):
            return self.list_files(path, limit, start, order, order_by)

        if result.get("code") == "0":
            return result.get("data", {})
        raise Exception(f"List files failed: {result.get('msg')}")

    def upload_file(self, file_path: str, target_path: str,
                   cover: bool = False) -> bool:
        """上传文件"""
        url = f"{self.base_url}/efile/openapi/v2/file/upload"
        headers = {"Authorization": f"Bearer {self.token}"}

        with open(file_path, 'rb') as f:
            files = {'file': f}
            data = {
                'path': target_path,
                'cover': 'cover' if cover else 'uncover'
            }

            response = self.session.post(url, headers=headers,
                                        files=files, data=data)
            result = self._handle_response(response)

            if result.get("retry"):
                return self.upload_file(file_path, target_path, cover)

            if result.get("code") == "0":
                return True
            raise Exception(f"Upload file failed: {result.get('msg')}")

    def download_file(self, file_path: str, save_path: str) -> bool:
        """下载文件"""
        url = f"{self.base_url}/efile/openapi/v2/file/download"
        headers = self._get_headers()

        params = {"path": file_path}
        response = self.session.get(url, headers=headers, params=params,
                                   stream=True)

        if response.status_code == 200:
            with open(save_path, 'wb') as f:
                for chunk in response.iter_content(chunk_size=8192):
                    if chunk:
                        f.write(chunk)
            return True
        raise Exception(f"Download file failed: HTTP {response.status_code}")

    # ===== 容器管理 API =====

    def create_container(self, container_config: Dict[str, Any]) -> str:
        """创建容器实例"""
        url = f"{self.base_url}/ai/openapi/v2/instance-service/task"
        headers = self._get_headers()

        response = self.session.post(url, json=container_config, headers=headers)
        result = self._handle_response(response)

        if result.get("retry"):
            return self.create_container(container_config)

        if result.get("code") == "0":
            return result.get("data")
        raise Exception(f"Create container failed: {result.get('msg')}")

    def list_containers(self, limit: int = 10, sort: str = "desc",
                       status: Optional[str] = None,
                       task_type: Optional[str] = None) -> List[Dict[str, Any]]:
        """查询容器实例列表"""
        url = f"{self.base_url}/ai/openapi/v2/instance-service/task"
        headers = self._get_headers()

        params = {
            "limit": limit,
            "sort": sort
        }
        if status:
            params["status"] = status
        if task_type:
            params["taskType"] = task_type

        response = self.session.get(url, headers=headers, params=params)
        result = self._handle_response(response)

        if result.get("retry"):
            return self.list_containers(limit, sort, status, task_type)

        if result.get("code") == "0":
            return result.get("data", [])
        raise Exception(f"List containers failed: {result.get('msg')}")

    def stop_containers(self, container_ids: List[str]) -> bool:
        """停止容器实例"""
        url = f"{self.base_url}/ai/openapi/v2/instance-service/task/actions/stop"
        headers = self._get_headers()

        params = {"ids": container_ids}
        response = self.session.post(url, headers=headers, params=params)
        result = self._handle_response(response)

        if result.get("retry"):
            return self.stop_containers(container_ids)

        if result.get("code") == "0":
            return True
        raise Exception(f"Stop containers failed: {result.get('msg')}")
```

## 常见问题

### Q1: 如何处理 Token 过期？

**A**: 当收到错误码 `10008` 时，表示 Token 已过期。需要：

1. 清除当前 Token
2. 重新调用登录接口获取新 Token
3. 重试之前的请求

客户端库通常会自动处理这个过程。

### Q2: 文件上传时如何覆盖已存在的文件？

**A**: 在上传文件时，设置 `cover` 参数：

- `cover=cover`: 覆盖已存在的文件
- `cover=uncover`: 如果文件已存在则跳过（默认）

### Q3: 如何下载文件夹？

**A**: 下载文件夹时，API 会自动将其压缩为 zip 格式返回。直接指定文件夹路径即可。

### Q4: 容器实例的状态有哪些？

**A**: 容器实例的状态包括：

- `Running`: 运行中
- `Deploying`: 部署中
- `Waiting`: 等待中
- `Terminated`: 已终止
- `Failed`: 失败
- `Completed`: 已完成

### Q5: 如何指定容器使用的 GPU？

**A**: 在创建容器时，设置以下参数：

- `acceleratorType`: 加速器类型（gpu、dcu、mlu、cpu）
- `gpuNumber`: GPU 卡数

### Q6: 作业的最大运行时间如何设置？

**A**: 使用 `GAP_WALL_TIME` 参数，格式为 `HH:MM:ss`，例如：

- `01:00:00`: 1 小时
- `12:30:45`: 12 小时 30 分 45 秒

### Q7: 如何在容器启动时执行初始化脚本？

**A**: 在创建容器时，设置：

- `useStartScript`: true
- `startScriptContent`: 脚本内容（使用 `\n` 表示换行）

### Q8: 如何挂载存储到容器？

**A**: 在创建容器时，配置 `mountInfoList`：

```json
"mountInfoList": [
  {
    "sourcePath": "/data",
    "targetPath": "/mnt/data",
    "accessMode": "rw"
  }
]
```

### Q9: 如何查询特定状态的容器？

**A**: 调用查询容器列表接口时，使用 `status` 参数过滤：

```
GET /ai/openapi/v2/instance-service/task?status=Running&limit=10
```

### Q10: 上传大文件时如何处理？

**A**:

1. 使用 multipart/form-data 格式
2. 分块上传（如果支持）
3. 实现重试机制处理网络中断

## 关键要点总结

1. **必须使用 HTTPS** - 确保数据安全
2. **Token 认证** - 除登录外所有接口都需要 Token
3. **错误处理** - 先检查 HTTP 状态码，再检查响应体中的 code 字段
4. **Token 过期处理** - 当 code=10008 时需要重新登录
5. **字符编码** - 统一使用 UTF-8
6. **多语言** - 通过 Accept-Language 请求头控制返回语言
